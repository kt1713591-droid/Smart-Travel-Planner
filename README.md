# ✈️ Smart Travel Planner (AI Based Project)

## Project Overview

The **Smart Travel Planner** is a rule-based AI application that suggests travel destinations based on user preferences such as **mood, budget, and purpose of travel**.

It uses a simple **knowledge-based system** to simulate decision-making similar to a human travel advisor.

---

## Objectives

* To design an intelligent system that recommends travel destinations
* To apply AI concepts like **agents, knowledge representation, and decision-making**
* To create an attractive **GUI using Python Tkinter**

---

## AI Concepts Used

* **Intelligent Agents** → Takes input and gives output
* **Knowledge Representation** → Database of places
* **Rule-Based System** → IF-ELSE decision logic
* **Problem Solving** → Suggesting best travel options

---

## Technologies Used

* **Programming Language:** Python
* **Library:** Tkinter (for GUI)
* **Concept:** Artificial Intelligence (Rule-Based System)

---

## Features

* Modern Dark Theme UI
* Select Mood, Budget, Purpose
* Smart Suggestions (AI logic)
* Database (India + International places)
* Multiple recommendations at once
* Interactive UI with hover effects

---

## Inputs

User selects:

* Mood → Relax / Adventure / Romantic / Spiritual / Party / Nature
* Budget → Low / Medium / High
* Purpose → Solo / Family / Friends

---

## Output

* Displays **top recommended travel destinations** based on selected inputs

---

## Working of the System

1. User selects mood, budget, and purpose
2. System checks the **knowledge base (places_db)**
3. Applies rule-based logic
4. Generates and displays suitable destinations
