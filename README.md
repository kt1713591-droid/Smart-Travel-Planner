# Travel Planner That Is Really Smart (Made With Artificial Intelligence)

## What This Project Is About

The Travel Planner That Is Really Smart is an application that uses rules to suggest places to visit based on what you like such as how you're feeling how much money you have and why you are traveling.

It uses a system to make decisions like a human who helps people plan their trips.

The Travel Planner That Is Really Smart is a tool for anyone who wants to travel.

---

## What We Want To Achieve

* Make a system that can suggest places to visit

* Use ideas from Artificial Intelligence like agents, knowledge and decision making

* Create a user interface that's nice to look at using Python Tkinter

* The user interface should be easy to use and understand

---

## Artificial Intelligence Ideas Used

* Intelligent Agents → These agents take what you say and give you an answer

* Knowledge → This is like a book of places

* Rule-Based System → This is like a set of rules that say if this happens then do that

* Problem Solving → This is like finding the best places for you to visit

## What You Need To Tell The Travel Planner

User selects:

* How you are feeling → You can choose from relax, adventure, romantic, spiritual, party or nature

* How money you have → You can choose from low, medium or high

* Why you are traveling → You can choose from solo, family or friends

## What The Travel Planner Gives You

* It shows you the places to visit based on what you told it

---

## How The Travel Planner Works

1. You tell the Travel Planner how you are feeling, how money you have and why you are traveling

2. The system looks at its book of places

3. It uses its rules to make a decision

4. It shows you the places to visit
